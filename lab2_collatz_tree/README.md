### 1. Условные конструкции: if, else, else if

В Rust `if` является **выражением**, а не инструкцией - он возвращает значение.

#### Базовый синтаксис

```rust
let number = 7;

if number < 5 {
    println!("меньше 5");
}
```

Условие всегда должно быть типа `bool` (неявных преобразований нет):

```rust
let number = 3;
if number {  // ОШИБКА! expected `bool`, found integer
    println!("истина");
}
```

#### if-else

```rust
let number = 7;

if number < 5 {
    println!("меньше 5");
} else {
    println!("больше или равно 5");
}
```

#### else if

```rust
let number = 6;

if number % 4 == 0 {
    println!("делится на 4");
} else if number % 3 == 0 {
    println!("делится на 3");
} else if number % 2 == 0 {
    println!("делится на 2");
} else {
    println!("не делится на 4, 3 или 2");
}
```

#### if как выражение

Поскольку `if` - выражение, его можно использовать справа от `let`:

```rust
let condition = true;
let number = if condition { 5 } else { 6 };

println!("number = {}", number);
```

**Важно:** обе ветки должны возвращать одинаковый тип!

```rust
let number = if condition {
    5
} else {
    "six"  // ОШИБКА! expected integer, found `&str`
};
```

### 2. Сопоставление с образцом: match

`match` - мощный инструмент для сопоставления с образцом (pattern matching).

#### Базовый синтаксис

```rust
let number = 13;

match number {
    1 => println!("Один"),
    2 => println!("Два"),
    3 => println!("Три"),
    _ => println!("Что-то другое"),  // _ - шаблон по умолчанию
}
```

#### match как выражение

```rust
let number = 5;

let description = match number {
    1 => "первый",
    2 | 3 | 5 | 7 | 11 | 13 => "простое число",  // несколько значений через |
    13 => "несчастливое число",
    _ => "обычное число",
};

println!("{}", description);
```

#### Сопоставление с диапазонами

```rust
let number = 42;

match number {
    1..=10 => println!("от 1 до 10"),
    11..=100 => println!("от 11 до 100"),
    _ => println!("больше 100 или меньше 1"),
}
```

#### Деструктуризация

```rust
let pair = (2, -2);

match pair {
    (0, y) => println!("x = 0, y = {}", y),
    (x, 0) => println!("x = {}, y = 0", x),
    (x, y) if x == -y => println!("x и y противоположны"),  // охранный шаблон (guard)
    _ => println!("нет совпадений"),
}
```

#### Match с Enum

```rust
enum Color {
    Red,
    Green,
    Blue,
    Rgb(u8, u8, u8),
}

let color = Color::Rgb(122, 17, 40);

match color {
    Color::Red => println!("Красный"),
    Color::Green => println!("Зелёный"),
    Color::Blue => println!("Синий"),
    Color::Rgb(r, g, b) => println!("RGB({}, {}, {})", r, g, b),
}
```

**Отличия match от if:**
- `match` проверяет все возможные варианты (exhaustive matching)
- `match` работает с шаблонами, а не только с bool
- Компилятор предупредит, если не все случаи обработаны

### 3. Циклы

Rust предоставляет несколько видов циклов: `loop`, `while`, `for`.

#### loop - бесконечный цикл

```rust
loop {
    println!("Бесконечный цикл!");
    // break для выхода
}
```

#### while - цикл с условием

```rust
let mut number = 3;

while number != 0 {
    println!("{}!", number);
    number -= 1;
}
println!("ПУСК!");
```

#### for - итерация по коллекциям и диапазонам

**Диапазон (Range):**

```rust
// от 1 до 4 (5 не включается)
for number in 1..5 {
    println!("{}", number);  // 1, 2, 3, 4
}

// от 1 до 5 (включая 5)
for number in 1..=5 {
    println!("{}", number);  // 1, 2, 3, 4, 5
}

// обратный порядок
for number in (1..=5).rev() {
    println!("{}", number);  // 5, 4, 3, 2, 1
}

// итерация с шагом (только нечётные)
for number in (1..=10).step_by(2) {
    println!("{}", number);  // 1, 3, 5, 7, 9
}

// каждое третье число
for number in (0..12).step_by(3) {
    println!("{}", number);  // 0, 3, 6, 9
}
```

**Итерация по коллекции:**

```rust
let fruits = ["яблоко", "банан", "апельсин"];

for fruit in fruits {
    println!("Фрукт: {}", fruit);
}
```

**Итерация с индексом:**

```rust
let fruits = ["яблоко", "банан", "апельсин"];

for (index, fruit) in fruits.iter().enumerate() {
    println!("{}: {}", index, fruit);
}
```

**Итерация по mutable коллекции:**

```rust
let mut numbers = [1, 2, 3, 4, 5];

for number in numbers.iter_mut() {
    *number *= 2;  // разыменование для изменения
}

println!("{:?}", numbers);  // [2, 4, 6, 8, 10]
```

#### Возврат значения из loop

`loop` может возвращать значение через `break`:

```rust
let mut counter = 0;

let result = loop {
    counter += 1;

    if counter == 10 {
        break counter * 2;  // возвращаемое значение
    }
};

println!("Результат: {}", result);  // 20
```

### 4. break и continue

#### break - выход из цикла

```rust
let mut i = 0;

loop {
    i += 1;
    if i == 5 {
        break;  // выход из цикла
    }
    println!("{}", i);
}
```

#### continue - переход к следующей итерации

```rust
for number in 1..=10 {
    if number % 2 == 0 {
        continue;  // пропускаем чётные
    }
    println!("{}", number);  // 1, 3, 5, 7, 9
}
```

#### Метки циклов (Loop Labels)

Для вложенных циклов можно использовать метки для указания, к какому циклу относится `break` или `continue`:

```rust
// Поиск простых чисел
'outer: for n in 2..=50 {
    for i in 2..n {
        if n % i == 0 {
            continue 'outer;  // переходим к следующей итерации внешнего цикла
        }
    }
    println!("{} - простое число", n);
}
```

```rust
// Выход из вложенных циклов
let mut count = 0;

'counting_up: loop {
    println!("count = {}", count);
    let mut remaining = 10;

    loop {
        println!("  remaining = {}", remaining);
        if remaining == 9 {
            break;  // выход из внутреннего цикла
        }
        if count == 2 {
            break 'counting_up;  // выход из внешнего цикла
        }
        remaining -= 1;
    }

    count += 1;
}
println!("Конец count = {}", count);
```

### 5. Блоки и видимость переменных

#### Блок как выражение

Блок `{ }` в Rust - это выражение, которое может возвращать значение:

```rust
let result = {
    let x = 5;
    let y = 10;
    x + y  // выражение без ; - возвращаемое значение блока
};

println!("result = {}", result);  // 15
```

#### Видимость переменных (Scope)

Переменные существуют только в пределах блока, где они объявлены:

```rust
let x = 1;

{
    let x = x + 1;  // новая переменная x (shadowing)
    println!("Внутри блока: x = {}", x);  // 2
}

println!("Снаружи блока: x = {}", x);  // 1
```

#### Возвращаемое значение блока

```rust
fn calculate_price(base_price: i32) -> i32 {
    let final_price = {
        let discount = 10;
        base_price - discount
    };  // блок возвращает base_price - discount

    // discount здесь недоступен!
    final_price
}
```

Блоки полезны для:
- Ограничения области видимости временных переменных
- Группировки связанных вычислений
- Возврата вычисленного значения

### 6. Макроопределения

Макросы позволяют писать код, который генерирует другой код во время компиляции.

#### Объявление макроса

```rust
// Простой макрос без аргументов
macro_rules! say_hello {
    () => {
        println!("Привет!");
    };
}

fn main() {
    say_hello!();  // разворачивается в println!("Привет!");
}
```

#### Макрос с аргументами

```rust
macro_rules! greet {
    ($name:expr) => {
        println!("Привет, {}!", $name);
    };
}

fn main() {
    greet!("Мир");      // Привет, Мир!
    greet!("Rust");     // Привет, Rust!
}
```

#### Несколько шаблонов

```rust
macro_rules! calculate {
    (add $a:expr, $b:expr) => {
        $a + $b
    };
    (sub $a:expr, $b:expr) => {
        $a - $b
    };
    (mul $a:expr, $b:expr) => {
        $a * $b
    };
}

fn main() {
    let x = calculate!(add 5, 3);   // 8
    let y = calculate!(sub 10, 4);  // 6
    let z = calculate!(mul 2, 7);   // 14
}
```

#### Повторение в макросах

```rust
macro_rules! vector {
    ( $( $x:expr ),* ) => {
        {
            let mut temp_vec = Vec::new();
            $(
                temp_vec.push($x);
            )*
            temp_vec
        }
    };
}

fn main() {
    let v = vector![1, 2, 3, 4, 5];
    println!("{:?}", v);
}
```

#### Обозначения для аргументов

| Обозначение | Тип | Пример |
|-------------|-----|--------|
| `expr` | Выражение | `2 + 2`, `"hello"` |
| `ident` | Идентификатор | `x`, `MyStruct` |
| `ty` | Тип | `i32`, `String` |
| `literal` | Литерал | `42`, `"text"` |
| `stmt` | Инструкция | `let x = 5;` |
| `pat` | Шаблон | `Some(x)` |
| `path` | Путь | `std::collections::HashMap` |
| `tt` | Token tree | любой токен |
| `block` | Блок | `{ ... }` |

#### Отличия макросов от функций

| Макросы | Функции |
|---------|---------|
| Разворачиваются при компиляции | Вызываются во время выполнения |
| Могут принимать переменное число аргументов | Фиксированное число аргументов |
| Могут генерировать код | Не могут генерировать код |
| Сложнее отлаживать | Легче отлаживать |
| `!` в конце имени | Без `!` |

#### Декларативные vs процедурные макросы

**Декларативные (`macro_rules!`):** описывают шаблоны сопоставления:

```rust
macro_rules! my_vec {
    ( $( $x:expr ),* ) => {
        vec![$($x),*]
    };
}
```

**Процедурные:** функции, работающие с AST (более мощные, но сложнее):

```rust
// В отдельном crate с #[proc_macro]
#[proc_macro]
pub fn make_answer(_item: TokenStream) -> TokenStream {
    "fn answer() -> u32 { 42 }".parse().unwrap()
}
```

## Условие задачи

### Последовательность Коллатца (гипотеза 3n+1)

Последовательность Коллатца для натурального числа n определяется правилами:

```
если n чётное:   n = n / 2
если n нечётное: n = 3n + 1
```

Последовательность продолжается, пока n не станет равным 1.

**Пример для n = 6:**
```
6 → 3 → 10 → 5 → 16 → 8 → 4 → 2 → 1
```
Длина последовательности: 9 (включая начальное число и 1)

**Пример для n = 11:**
```
11 → 34 → 17 → 52 → 26 → 13 → 40 → 20 → 10 → 5 → 16 → 8 → 4 → 2 → 1
```
Длина последовательности: 15

### Задание

1. Реализуйте функцию `collatz_length(n: u64) -> u32`, которая возвращает длину последовательности Коллатца для числа n
2. Реализуйте функцию `collatz_sequence(n: u64) -> Vec<u64>`, которая возвращает всю последовательность
3. Найдите число от 1 до 1_000_000 с самой длинной последовательностью Коллатца

### Требования

- Используйте цикл `loop` или `while` для вычисления последовательности
- Используйте `match` или `if` для проверки чётности
- Программа должна запрашивать число у пользователя и выводить последовательность и её длину
- Для поиска числа с максимальной длиной используйте цикл `for`

### Пример работы программы

```
Введите число: 6
Последовательность: 6 → 3 → 10 → 5 → 16 → 8 → 4 → 2 → 1
Длина: 9

Число с максимальной длиной последовательности (1-1000000): 837799
Длина: 525
```

### Подсказки

- Для проверки чётности используйте `% 2` или метод `.is_even()` (для целых чисел)
- Тип `u64` достаточен для чисел до 1_000_000 (переполнения не будет)
- Гипотеза Коллатца до сих пор не доказана математически, но проверена для всех чисел до очень больших значений
